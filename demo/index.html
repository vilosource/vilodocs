Panes Demo - Keyboard Shortcuts

  • Ctrl+\ Split horizontally
  • Ctrl+Alt+\ Split vertically
  • Ctrl+W Close tab
  • Ctrl+Tab Next tab
  • Ctrl+Shift+Tab Previous tab
  • Ctrl+B Toggle sidebar
  • Ctrl+J Toggle panel
  • Ctrl+K Ctrl+W Close all tabs
  • F6 Focus next pane
  • Shift+F6 Focus previous pane

Drag tabs to reorder or drop on edges to create splits!
